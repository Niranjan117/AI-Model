REQUIRED API KEYS:

1. OPENWEATHERMAP API (Weather Data)
   - Website: https://openweathermap.org/api
   - Sign up for free account
   - Get API key from dashboard
   - Replace: self.weather_api_key = "your_openweather_api_key"
   - Provides: Temperature, humidity, wind, pressure, weather conditions

2. NASA API (Soil Moisture & Satellite Data)
   - Website: https://api.nasa.gov/
   - Sign up for free account
   - Get API key instantly
   - Replace: self.nasa_api_key = "your_nasa_api_key"
   - Provides: Soil moisture, satellite imagery metadata

3. SENTINEL HUB API (Satellite Images)
   - Website: https://www.sentinel-hub.com/
   - Sign up for account
   - Get OAuth token
   - Replace: self.sentinel_api_key = "your_sentinel_api_key"
   - Provides: High-resolution satellite imagery, cloud coverage

4. USGS EARTH EXPLORER (Optional - Landsat Data)
   - Website: https://earthexplorer.usgs.gov/
   - Free registration
   - Access to Landsat satellite data
